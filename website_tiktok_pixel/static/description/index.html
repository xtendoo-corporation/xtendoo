Odoo TikTok integration - activate pixel to find new customers, optimize your campaigns and measure ad performance

Description

Activate the Tiktok Pixel script to track website visitors and build Odoo TikTok integration with Ads Manager.

This module allows adding a pixel with the "Manually install pixel code" approach. After installation and adding a pixel code, the website visitor action "view page" will be tracked for each website page.

TikTok web attribution through the pixel allows you to measure your TikTok Ads effectiveness, to found out the ROAS (Return on Ad Spend) metric.

Read more about Tiktok Pixel: https://ads.tiktok.com/help/article/tiktok-pixel

Configuration
Adding a tracking code
Go to the menu "Website" - "Configuration" - "Settings" and activate the "TikTok Pixel" option. Then enter a Tracking Code.
Apply the changes.
Results
The Tiktok Pixel will be activated on all pages of the website.
  • The TikTok Pixel script will not be activated, if you are logged in to Odoo and you have Website editor's access rights.
  • This behaviour is implemented to prevent sending data from internal users.
  • Log out from the system or use the incognito mode or other browser to test Pixel script.
Testing
