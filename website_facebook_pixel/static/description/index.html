Integrate Facebook Pixel to an Odoo website and track customer visits

Description

Add the Meta Pixel code to your Odoo website and get analytic data that could be used for understanding visitor's actions. With this script, you get ability to track Website Activities, Improve Your Return on Advertising, Reach New and Existing Customers on Facebook Ads.

To track Odoo Facebook Pixel eCommerce events, such as "ViewContent", "Search", "AddToWishlist", "AddToCart", "InitiateCheckout", "AddPaymentInfo", and "Purchase" on your Odoo website, consider using of the app: Odoo Facebook Pixel eCommerce Tracking.

Configuration
Adding a tracking ID
Go to the menu "Website" - "Configuration" - "Settings" and activate the "Facebook | Meta Pixel" option. Then enter Tracking ID.
Apply the changes.
Testing
Testing by the Chrome extension

You can easily test the Odoo Facebook Pixel integration via the Chrome extension "Meta Pixel Helper".

Download it by this link: https://chrome.google.com/webstore/detail/meta-pixel-helper/fdgfkebogiimcoedlicjlajpkdmockpc

Found out the testing details in our YouTube manual.

Testing Particularities

As the Meta Pixel script does not have to send tracking data about a website activity of internal users, there are several testing tips:

  • You will not see the Meta Pixel script if you are logged in to Odoo and you have internal user's access rights.
  • Log out from the system or use the incognito mode or other browser to test Pixel script.
