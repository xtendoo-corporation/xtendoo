Overview

Whatsapp is an immensely popular chatting app used by 1.5 Billion people worldwide. It has an easy interface and can be used powerfully with Odoo. Pragmatic has developed an Odoo app which allows users to use the Whatsapp Application to send messages via Odoo. We can send messages from Contacts, Sales, Accounts invoice, Accounts Payments, Credit Notes, Delivery orders, Point of sale, Purchase orders, Project Task, CRM Lead, Payment Reminder, User signup page via the same application. Let us have a look at how this works inside Odoo.

Configuration

It is difficult to get a signup for WhatsApp for business directly and get access to the API as the signup process takes too long. Hence, it is better to sign up for third party whatsapp API services such as Chat API. Also selenium could be used for free but it is not robust and reliable and cannot handle large volumes of load.



You need to signup for a Chat Api account:

Copy Link
https://bit.ly/3MFXxY0


Click on “Add new Instance” to create an instance.

Click on create Instance. Fill all mandatory fields to create an account.

After an account is created on the chat api.
We get a whatsapp instance endpoint & whatsapp token.

If you create a free account then your account will expire in 3 days after that you need to purchase a subscription

If you scan QR code from chat api no need to scan QR code from odoo & vice versa.

Copy whatsapp endpoint (without last “/”) & whatsapp token.

In odoo, go to General Settings and Paste the credentials for Whatsapp Instance Endpoint (without last “/”) and Whatsapp token copied from whatsapp chat api.

If you scan QR code from chat api no need to scan QR code from odoo & vice versa.

Click on Get QR Code & scan QR code from mobile Whatsapp Web.

After you scan QR code from your mobile is authenticated

Following conditions need to rescan QR code:

  1. If the QR code is not scanned within 45 seconds, the QR code will expire.
  2. If whatsapp endpoint or whatsapp token is not added correctly & not getting QR code.
  3. If the authentication fails/accidently wizard is closed then you need to click on the logout button after that click on Get QR code.

Configuration of bidirectional messages (send & receive messages from whatsapp in Odoo):

In odoo activate developer mode Goto Technical → System Parameters → web.base.url → Copy Value

Chat API Configuration:-

In the webhook url you need to paste the value (copied from odoo web.base.url) after that add “/whatsapp/responce/message”

Webhook url : base url + /whatsapp/responce/message

Example:

https://staging.pragtech.co.in/whatsapp/responce/message

  • Base url: https://staging.pragtech.co.in/
  • Controller path : /whatsapp/responce/message

In the configuration of the chat API refer to the below example & enable all fields (Acknowledge(ACK), Notification of changes in chats, Notification of changes in status, Not send old messages after authorization, Turn on video uploading ) & click on save.
