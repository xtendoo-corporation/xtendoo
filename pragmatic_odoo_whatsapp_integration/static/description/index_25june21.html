Configuration
It is difficult to get a signup for WhatsApp for business directly and get access to API as signup process takes too long. Hence, it is better to signup for third party whatsapp API services as Chat API. Also selenium could be used for free but it is not robust and reliable and cannot handle large volumes of load. You need to signup for Chat Api account on https://app.chat-api.com/login

First we have to login into whatsapp chat api using https://app.chat-api.com/login.
We get an whatsapp instance id & whatsapp token.



Go to General Settings and Paste the credentials for Whatsapp Instance Id and Whatsapp token copied from whatsapp chat api.
